Spending Tracker App

Take control of your finances with powerful transaction analytics

About This Project

The Spending Tracker is a React-based financial transaction analyzer that helps users track, categorize, and visualize their spending patterns with advanced monthly comparison and recurring transaction insights.

Multi-file Transactions

Upload and analyze transactions from multiple sources simultaneously to get a complete picture of your finances.

Custom Categories

Create custom categories and mark them as fixed or flexible expenses to better understand your spending habits.

Monthly Comparisons

Compare your spending month-over-month to identify trends and areas where you can save money.

Recurring Payment Detection

Automatically identify subscription services and recurring payments to help you manage your regular expenses.

Key Features

  • Interactive dashboard with comprehensive spending analytics
  • Transaction management with category editing and flagging capabilities
  • Multi-file support for analyzing data from different sources
  • Flexible date filtering to focus on specific time periods
  • Modern, responsive UI built with React and Tailwind CSS

Technologies Used

  • React.js with TypeScript
  • Vite for fast development and building
  • Tailwind CSS with shadcn/ui components
  • Chart visualization with Recharts
  • Node.js/Express backend
